As part of this course, you'll learn how to utilize IBM Integration Bus to link applications, regardless of the message formats or protocols they support.

Overview

  This course will show you how to create, deploy, and maintain message flow applications using IBM App connect Enterprise. These applications employ a variety of messaging topologies to convey messages between service requesters and service providers, as well as to route, modify, and enhance messages as they are processed.
   This course teaches you how to create data transport and transformation apps. The course also covers how to utilize databases and maps to modify and enhance data during processing, as well as how to regulate the flow of data using various processing nodes. You'll also learn how to use the Data Format Description Language to create data models (DFDL).
